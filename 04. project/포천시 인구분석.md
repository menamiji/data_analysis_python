- 사용할 행 지정


- CSV 파일 불러오기
```python

```

- 사용할 인덱스 지정
```python

```

```python
df = df.reset_index()
```

- 데이터 프레임을 모두 정수로 설정

## 2-2. 그래프 그리기
### 막대그래프
```python
plt.bar(df.index.tolist(), df['소흘읍'])
plt.show()
```
- df.index.tolist() : x축의 값을 나이로 지정 (0세 ~ 110세)
- df['소흘읍'] : y축의 값을 소흘읍의 인구수로 지정

### 그래프 사이즈 조절
